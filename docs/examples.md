# Examples

Example inputs and sample outputs live in `examples/`.

## Inputs

- `input_expert_landing_ru.json`
- `input_psychologist_ru.json`
- `input_author_site_ru.json`

## Sample outputs

- `output_expert_landing_ru.md`
- `output_psychologist_ru.md`
- `output_author_site_ru.md`
- `output_mock_ru.md`

Each example is designed to test whether the assistant explains typography through:

```text
visible feature -> perception effect -> layout role
```

The exact font pair is less important than the reasoning quality: the answer should explain visible features, perception effect, layout role, readability, Cyrillic checks, and risks.
