# Limitations

This MVP is intentionally small.

## Not included

- embeddings;
- vector database;
- account system;
- saved project history;
- production-grade font licensing audit;
- visual design scoring;
- PDF export;
- screenshot rendering;
- self-hosted font files;
- platform-specific verification for Tilda, Webflow, Framer, Figma, or similar tools.

## Important boundaries

Google Fonts metadata can indicate whether a family exists, which subsets are listed, and which variants or axes are available. It does not guarantee high-quality Cyrillic design.

The LLM recommendation is a starting point for design judgment, not a final visual approval.

The assistant should not claim platform-specific behavior unless that behavior is actually verified by the implementation.
