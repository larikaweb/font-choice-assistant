# Example output: psychologist website, Russian

## Project impression

The typography should feel calm, warm, and professional. It should not become childish, overly feminine, or decorative.

## Base font direction

Use a human or soft sans-serif as the base. The main text needs comfortable reading and a gentle tone.

## Main recommendation

- One-font safe system: **Manrope**
- Optional softer alternative: **Lora** for headings + **Manrope** for body text

## What the fonts do visually

- **Manrope**: rounded details and clean proportions -> softens the interface while keeping it structured -> works as the main family for body, buttons, labels, and headings in a one-font system.
- **Lora**: serif texture with a literary rhythm -> adds a more personal, reflective tone -> works only as a short heading or quote accent if the site needs more warmth.

## Cyrillic check

Test:

```text
Психология
Консультация
Программа
Заявка
Ж Д У Щ Я ф
ст ск пр ия нт ция
```

Check that headings do not feel too decorative and body text does not become too pale or narrow.

## Avoid

Avoid script fonts for warmth, very thin weights, and a second accent font if Manrope alone already gives enough softness and hierarchy.
