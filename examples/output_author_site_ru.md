# Example output: fantasy author site, Russian

## Project impression

The site should feel literary, atmospheric, and slightly mysterious, while keeping reading comfortable.

## Base font direction

Use a readable serif or a calm sans-serif base depending on text density. If the page has long book descriptions, body readability must come first.

## Main recommendation

- Heading: **Cormorant Garamond**
- Body text and buttons: **Inter** or **Manrope**

## What the fonts do visually

- **Cormorant Garamond**: high contrast, elegant serifs, and a dramatic literary rhythm -> creates atmosphere and a bookish voice -> works as a heading or quote accent, not for dense interface text.
- **Inter / Manrope**: neutral spacing and stable sans-serif forms -> prevent the page from becoming too ornate -> work for body text, buttons, navigation, and metadata.

## Cyrillic check

Test:

```text
Архитектура
Программа
Заявка
Психология
Ж Д У Щ Я ф
ст ск пр ия нт ция
```

For the fantasy tone, especially check whether `Ж`, `Д`, and `Щ` feel elegant rather than awkward or too fragile in large headings.

## Avoid

Avoid using the atmospheric serif everywhere. The pair works because one font creates character and the other keeps the reading path clear.
