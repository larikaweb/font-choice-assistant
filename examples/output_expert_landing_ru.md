# Example output: expert landing page, Russian

## Project impression

The page should feel calm, structured, professional, and human rather than bureaucratic.

## Base font direction

Use a readable neutral or softly polished sans-serif for the main text. The base font should make scanning easy and keep long blocks comfortable.

## Main recommendation

- Heading: **Lora**
- Body text, buttons, captions: **Manrope**

## What the fonts do visually

- **Lora**: soft serif rhythm and book-like letterforms -> adds a calm authorial layer without becoming decorative -> works as a heading or quote accent.
- **Manrope**: clean sans-serif construction, open spacing, and practical weight range -> keeps text clear and modern without sounding cold -> works for body text, buttons, cards, and navigation.

## Cyrillic check

Test the pair on:

```text
Консультация
Документы
Стоимость
Образование
Архитектура
Ж Д У Щ Я ф
ст ск пр ия нт ция
```

Check whether Lora headings remain calm rather than too literary, and whether Manrope body text stays readable in dense sections.

## Avoid

Avoid a very decorative heading font or two similar sans-serifs without a clear role split. Do not claim that either font is built into a specific platform unless that is verified.
